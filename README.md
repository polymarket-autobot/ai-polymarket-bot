## Core Features

- Multiple operation modes — Run (live trading), TEST (simulation), Monitor (hands-free position management)  
- Multilingual documentation — English, Russian, German, Spanish, French, Portuguese, Italian, Ukrainian, Polish, Chinese

## Quick Start

**Requirements**  
Python 3.10 or higher

# 1. Install dependencies
pip install -r requirements.txt

# 2. Launch the application
python main.py


**Initial Setup (Required)**

1.  Navigate to **Settings**
    -   Enter your **Private key**
    -   Enter your **Polymarket proxy address**
2.  Navigate to **API AI**
    -   Select provider (Groq / Google / xAI / OpenRouter / Anthropic / OpenAI)
    -   Input API key
    -   Choose model
    -   Run **Test connection** to verify
3.  (Optional) Navigate to **PROMPT** to customize system prompts for each strategy

Full multilingual instructions → open Readme.html in your browser.
